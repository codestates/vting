## [Vting 바로가기](v-ting.net)

## ABOUT
<br/>

" 모두가 참여하는 실시간 설문조사 <br/>
쉽고, 직관적이고, 재미있게! <br/>
실시간으로 결과 또한 확인해 보세요."

<br/>

<b> Vting 에서는 </b>

📳 핸드폰 및 웹으로 투표에 참여할 수 있어요 <br/>
✍ 투표에 바로 참여할 수 있도록 QR code 또는 웹서비스 내의 code를 제공해요 <br/>
✅ 다양한 방식으로 투표 결과를 만들 수 있어요 <br/>
💬 소셜 로그인 기능이 제공돼요 <br/>
📶 결과를 실시간으로 확인할 수 있어요 <br/>
😜 귀여운 애니메이션 효과로 재밌게 결과를 지켜볼 수 있어요 <br/>

<br/>
